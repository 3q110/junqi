# 军旗人机对战 (Jiqi - Chinese Military Chess)

终端版军旗游戏，玩家 vs 电脑（简单随机AI）。

## 规则

- **棋盘**: 7×11，中间为河流（第5行）
- **棋子**: 每方24子（军长、师长、旅长、团长、营长、连长、排长、工兵、地雷×3、炸弹×3、司令、军旗）
- **玩法**: 暗棋模式，棋子隐藏，面对面才知身份
- **胜负**: 吃掉对方军旗即胜

### 棋子等级

| 等级 | 棋子 | 数量 |
|------|------|------|
| 1 | 军长 | 1 |
| 2 | 师长 | 1 |
| 3 | 旅长 | 2 |
| 4 | 团长 | 2 |
| 5 | 营长 | 2 |
| 6 | 连长 | 2 |
| 7 | 排长 | 2 |
| 8 | 工兵 | 3 |
| 9 | 地雷 | 3 |
| 10 | 炸弹 | 3 |
| 11 | 军旗 | 1 |

### 特殊规则

- **工兵**可拆地雷
- **炸弹**与任何棋子相撞同归于尽
- **地雷**非工兵/炸弹触碰则被炸死
- **军旗**不能移动，被吃掉即输
- 棋子只能上下左右移动一格

## 安装

```bash
# 无需额外依赖，纯 Python
cd jiqi
```

## 运行

```bash
python game.py
```

## 测试

```bash
python -m pytest tests/ -v
```

## 项目结构

```
jiqi/
├── __init__.py    # 包初始化
├── piece.py       # 棋子类与类型定义
├── board.py       # 棋盘网格管理
├── rules.py       # 移动验证与战斗解析
├── player.py      # 人类玩家 & AI玩家
├── ui.py          # 终端界面渲染
├── game.py        # 游戏主循环与入口
└── README.md      # 本文件

tests/
├── __init__.py
├── test_piece.py       # 棋子单元测试
├── test_board.py       # 棋盘单元测试
├── test_rules.py       # 规则引擎单元测试
├── test_player.py      # 玩家逻辑单元测试
├── test_ui.py          # 界面单元测试
├── test_game.py        # 游戏主循环单元测试
└── test_integration.py # 集成测试
```

## 技术栈

- **语言**: Python 3.11+
- **测试**: pytest
- **依赖**: 无（纯标准库）

## 开发

采用 TDD（测试驱动开发），所有功能均有对应测试。

```bash
# 运行全部测试
python -m pytest tests/ -v

# 运行单个测试文件
python -m pytest tests/test_rules.py -v
```

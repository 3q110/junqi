## 游戏说明

### 基本规则

军旗是中国经典策略棋类游戏，两人对弈（本版本为玩家 vs 电脑）。双方各执 24 枚棋子，目标是在不暴露己方军旗身份的情况下，通过策略移动和战斗，最终吃掉对方的军旗。

### 棋盘

- **尺寸**: 7 列 × 11 行
- **河流**: 第 5 行（中间横贯整盘的空白区域）
- **布局**: 己方棋子占据上方 5 行（0-4 行），电脑棋子占据下方 5 行（6-10 行）

### 棋子一览

| 符号 | 名称 | 数量 | 说明 |
|------|------|------|------|
| 司 | 军长 | 1 | 场上最大（除司令外） |
| 军 | 司令 | 1 | 场上最大 |
| 师 | 师长 | 2 | |
| 旅 | 旅长 | 2 | |
| 团 | 团长 | 2 | |
| 营 | 营长 | 2 | |
| 连 | 连长 | 2 | |
| 排 | 排长 | 2 | |
| 工 | 工兵 | 3 | 唯一可安全拆除地雷的棋子 |
| 炸 | 炸弹 | 3 | 与任何棋子相撞，双方同归于尽 |
| 地雷 | 地雷 | 3 | 不可移动，非工兵/炸弹触碰则被炸毁 |
| 旗 | 军旗 | 1 | 不可移动，被吃掉即输 |

### 棋子等级排序

司令 > 军长 > 师长 > 旅长 > 团长 > 营长 > 连长 > 排长 > 工兵

### 特殊规则

1. **工兵拆地雷**: 工兵可以安全地移动到地雷位置，将其拆除
2. **炸弹同归于尽**: 炸弹与任何棋子（包括地雷、军旗）相撞，双方同时消失
3. **地雷防御**: 地雷不能被除工兵和炸弹以外的任何棋子吃掉
4. **军旗不可动**: 军旗只能原地不动，被对方棋子走到位置上即被吃掉
5. **移动方式**: 每回合只能上下左右移动一格（不能斜走）
6. **暗棋模式**: 敌方棋子显示为 `?`，只有碰撞时才揭示身份

### 胜利条件

- 吃掉对方军旗 → **获胜**
- 己方军旗被吃掉 → **失败**

## 安装运行

### 本地运行（无需安装依赖）

```bash
git clone https://github.com/3q110/junqi.git
cd junqi/jiqi
python game.py
```

### 在线试玩

直接访问: https://3q110.github.io/junqi/

## 操作指南

1. **选择棋子**: 输入 `列,行` 坐标（如 `3,0` 表示第 3 列第 0 行）
2. **选择目标**: 输入目标位置坐标
3. **退出游戏**: 输入 `q`
4. **取消选择**: 输入 `c`

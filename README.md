# 军棋人机对战 (Junqi / Chinese Military Chess)

> 经典军棋终端游戏，支持人机对战。

## 在线试玩

直接在浏览器中打开 `index.html` 即可游玩，无需服务器：

- 双击 `index.html` 文件
- 或通过 GitHub 直接查看：[在线试玩](https://github.com/3q110/junqi/blob/main/index.html)（右键 → "Open in new tab" 可交互）

## 游戏规则

### 棋盘
- 棋盘为 5×9 网格（不含界河），双方各占一侧
- 中间为"界河"，上方为铁路线
- 己方底线两侧为两个"大本营"（不可移动棋子）

### 棋子（每方 24 枚）

| 棋子 | 数量 | 说明 |
|------|------|------|
| 司令 | 1 | 最大军衔，可吃除炸弹、地雷外所有棋子 |
| 军长 | 1 | |
| 师长 | 1 | |
| 旅长 | 1 | |
| 团长 | 1 | |
| 营长 | 1 | |
| 连长 | 1 | |
| 排长 | 1 | |
| 工兵 | 1 | 可挖地雷，可修铁路 |
| 炸弹 | 3 | 与任何棋子同归于尽（含司令） |
| 地雷 | 3 | 只能放在底线，不可移动，炸死除工兵外所有棋子 |
| 军旗 | 1 | 被吃掉则判负，不可移动 |

### 胜负判定
- 吃掉对方军旗 → 胜
- 对方无棋可走 → 胜
- 己方军旗被吃 → 负

### 移动规则
- 普通棋子：每次走一步（上下左右）
- 铁路线上的棋子：可沿铁路走任意格
- 工兵：可走曲线路径

### 战斗规则
- 高军衔吃低军衔
- 同军衔相撞 → 同归于尽
- 炸弹 → 与任何棋子同归于尽
- 地雷 → 炸死除工兵外所有棋子（地雷也被毁）
- 工兵 → 可挖地雷（工兵存活，地雷被毁）
- 军旗/大本营 → 不可移动，被吃即输

## 安装与运行

```bash
# 需要 Python 3.9+
pip install -e .

# 启动游戏
junqi
```

## 项目结构

```
junqi/
├── jiqi/            # 游戏核心
│   ├── piece.py     # 棋子定义
│   ├── board.py     # 棋盘逻辑
│   ├── rules.py     # 战斗规则
│   ├── ai.py        # AI 对手
│   ├── player.py    # 玩家操作
│   ├── ui.py        # 终端界面
│   └── game.py      # 游戏主循环
├── tests/           # 单元测试 (37 tests)
├── index.html       # 浏览器在线版
└── pyproject.toml
```
